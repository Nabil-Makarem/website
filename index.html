Prozessbeschreibung

Diese Station dient zür Lagerung und Zuführung von Werkstücken.

1. Durch das Drücken vom Start Taster wird ein Bauteil auf das Band von eienem Zylinder freigegeben.

2. Das Laufband fließt bis das Bauteil zum Ausgangslichtsensor gebracht wird.

3. Jetzt ist der Prozess fertig und kann erneut gestartet werden durch das Drücken vom Start Taster.
